This application currently only works on desktop.
Add participants to participants-example.js and rename it participants.js
The participants whose amnesty status is set to false are by default on the board.
Amnesty status for the round can be altered in the table on the left, but this doesn't alter the participants file.
Push the start button to select a random person among the people whose amnesty is set to false.

To run:
Download the code and unzip it in a folder on your computer.
Open a browser and navigate to that folder. Open index.html. Done!
