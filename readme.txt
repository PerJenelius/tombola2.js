Add participants by writing their name in the box in the bottom left and clicking "Add/Remove".
Clicking the button while the name of an existing user is entered removes that user from the list.
The participants whose amnesty status is set to false are entered onto the board.
Amnesty status for each participant can be altered by checking/unchecking their checkbox.
Push the start button to select a random person among the people whose amnesty is set to false.
The data is saved in localstorage in your browser for future use.

To run:
Download the code and unzip it in a folder on your computer.
Open a browser and navigate to that folder. Open index.html. Done!
